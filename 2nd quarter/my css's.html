MY CLASSES AND ID EXAMPLE

The class attribute is often used to point to a class name in a style sheet. It can also be used by a JavaScript to access and manipulate elements with the specific class name.

The id attribute is used to point to a specific style declaration in a style sheet. It can also be used by a JavaScript to access and manipulate a specific element with the specific id.
